# 🌤️ WeatherNow — Live Weather App
 
A clean and responsive weather application that shows real-time current weather and 5-day forecast for any city or town worldwide.
 
🌐 **Live App:** https://gayathrideshaveni.github.io/Weather-App/
 
---
 
## ✨ Features
 
- 🔍 Search weather for any city or town worldwide
- 🌡️ Current temperature, feels like, humidity, wind speed, visibility
- 📅 5-day weather forecast with daily breakdown
- 🌧️ Dynamic weather icons (sun, clouds, rain, snow, etc.)
- ⌨️ Search by pressing Enter or clicking Search button
- ❌ Error handling for invalid city names
- 📱 Responsive design — works on mobile and desktop

## 🚀 How to Run Locally
 
```bash
# 1. Clone the repository
git clone https://github.com/GayathriDeshaveni/Weather-App.git
cd Weather-App
 
# 2. Open index.html in your browser
# Just double-click index.html — no server needed!
```
 
> To use your own API key, sign up at [openweathermap.org](https://openweathermap.org/api) and replace the `apiKey` variable in `app.js`

## 🔌 APIs Used
 
- **Current Weather:** `api.openweathermap.org/data/2.5/weather`
- **5-Day Forecast:** `api.openweathermap.org/data/2.5/forecast`
Both APIs are from OpenWeatherMap — free tier, no credit card needed.
